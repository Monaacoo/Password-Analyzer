Ferramenta de análise e simulação de segurança de senhas desenvolvida em Python.

O projeto avalia a força de senhas, identifica padrões inseguros e simula ataques reais utilizando wordlists conhecidas (como rockyou.txt), demonstrando na prática como credenciais fracas podem ser comprometidas.

##  Funcionalidades

- Análise de força de senha (score e classificação)
- Detecção de padrões comuns (sequências, senhas populares, etc.)
- Estimativa de tempo de quebra (brute force teórico)
- Simulação de ataque por dicionário (wordlist)
- Suporte a hash seguro (bcrypt)
- Geração de relatório em JSON

## Objetivo

Demonstrar conceitos de segurança ofensiva e conscientização sobre boas práticas de criação de senhas, aproximando teoria e prática em cenários reais.

## Tecnologias

- Python
- bcrypt
- JSON
- Wordlists (rockyou)

## Aviso

Este projeto é destinado exclusivamente para fins educacionais e testes em ambientes controlados.
